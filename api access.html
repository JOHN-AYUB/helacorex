Authentication

All requests require a Bearer token in the Authorization header. Obtain your key from the Helacore Dashboard.

Header Format
Authorization: Bearer hk_live_abc123def456

Never expose your API key in client-side code. Use environment variables or a backend proxy.

API Endpoints

Complete reference for all available API endpoints. Click any endpoint to expand details.

GET /health-score Retrieve business health score

Returns the current Financial Health Score (0-100) and its component breakdown for the authenticated business.

Query Parameters

ParamTypeRequiredDescription
periodstringNotoday, week, month, quarter. Default: today
include_componentsbooleanNoInclude breakdown by component. Default: true

Response

{
  "score": 87,
  "period": "today",
  "components": {
    "cash_flow": 92,
    "profitability": 84,
    "debt_management": 78,
    "inventory_efficiency": 91,
    "growth_trajectory": 82
  },
  "trend": "improving",
  "last_updated": "2024-01-15T00:00:00Z"
}
GET /recommendations Get AI recommendations

Returns personalized AI-generated recommendations ordered by priority and expected impact.

Query Parameters

ParamTypeRequiredDescription
categorystringNoFilter: inventory, pricing, debt, growth, all
prioritystringNoFilter: urgent, high, medium, low
limitintegerNoMax results. Default: 10, max: 50
POST /simulate Run What-If simulation

Run a What-If simulation to predict the outcome of a business decision before implementing it.

Request Body

FieldTypeRequiredDescription
typestringYesprice_change, loan_impact, inventory_expansion, staffing
parametersobjectYesSimulation-specific parameters (varies by type)
duration_daysintegerNoSimulation period. Default: 30

Example: Price Change

{ "type": "price_change", "parameters": { "change_percentage": 10, "affected_products": ["maize_flour", "cooking_oil"] }, "duration_days": 30 }
POST /transactions Record a transaction

Record a business transaction (sale, expense, loan payment, or savings deposit).

Request Body

FieldTypeRequiredDescription
typestringYessale, expense, loan_payment, savings
amountnumberYesTransaction amount in KES
descriptionstringNoFree-text description
itemsarrayNoLine items with name, quantity, unit_price
timestampISO 8601NoDefault: current time
GET /market-insights Access market intelligence

Access anonymized, aggregated market intelligence data for your region and business category.

Query Parameters

ParamTypeRequiredDescription
regionstringNoCounty or area. Default: user's registered location
categorystringNoBusiness category filter
insight_typestringNotrends, seasonal, pricing, supply_chain
PUT /webhooks Configure webhooks

Register or update webhook URLs to receive real-time event notifications.

Request Body

FieldTypeRequiredDescription
urlstringYesHTTPS endpoint URL
eventsarrayYesEvent types: health_score.changed, recommendation.new, alert.critical
secretstringNoHMAC signing secret for verification

Error Codes

Status Code Description Resolution
200 OK Request successful —
400 BAD_REQUEST Missing or invalid parameters Check request body against schema
401 UNAUTHORIZED Invalid or expired API key Verify your Bearer token
404 NOT_FOUND Resource or endpoint not found Check URL and parameters
429 RATE_LIMITED Too many requests Wait for Retry-After header value
500 SERVER_ERROR Internal server error Retry with exponential backoff
